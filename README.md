# new-ai-summarizer-with-local-storage-
 Chrome Extension that instantly summarizes any article using Google’s Gemini AI API. Extract key information from web articles in multiple formats: Brief summaries, Detailed analysis, Bullet points and stores the history on local storage.

To Run it on your device:
Download the file and add your API key in the **content.js** file where mentioned.
Open your chrome, set it to developer mode and load the entire file
